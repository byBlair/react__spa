# react로 구현한 설화수 웹페이지

## 소개

리액트를 이용하여 전체적인 웹페이지 구조를 짜고 구현한 사이트입니다.

## 배포링크

배포방식은 AWS S3의 간이 배포를 이용하였습니다.  

## 개발 도구 및 환경

1. 개발환경 : VSCode,npm
2. 개발도구 : HTML,SCSS,react,react-dom,redux,react-router-dom,swiper
